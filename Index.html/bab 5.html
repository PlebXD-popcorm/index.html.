Bab 5: kesimpulan

Setelah berhasil merakit dan menguji rangkaian fire sensor dan water level sensor, dapat disimpulkan bahwa sistem ini bekerja dengan baik dalam mendeteksi keberadaan api dan tingkat ketinggian air. Fire sensor mampu memberikan respons cepat terhadap keberadaan api melalui perubahan suhu atau cahaya, sedangkan water level sensor dapat memberikan informasi akurat mengenai ketinggian air pada berbagai level. Rangkaian ini sangat bermanfaat untuk sistem keamanan dan pemantauan lingkungan, terutama untuk mencegah bahaya kebakaran. Proses perakitan juga memperkuat pemahaman tentang sensor, komponen elektronik, serta cara kerja sistem monitoring otomatis.
